# Ralph

Ralph is a tool for managing AI agent loops and pin operations.

## Project Structure

This repository is split into two main components:

- **[ralph_legacy](./ralph_legacy)**: A script-only version of Ralph. It contains legacy shell and Python scripts used for building specs, validating pins, and running agent loops.
- **[ralph_tui](./ralph_tui)**: A more robust TUI (Terminal User Interface) version written in Go. This version is currently "in flight" and aims to provide a more integrated and powerful experience for managing Ralph's operations.

## Getting Started

Refer to the individual READMEs in each directory for specific instructions on how to use each version.
